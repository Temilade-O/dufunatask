Software Development Process

Organized by Dufuna.Org

Software development is the process of developing software through successive phases in an orderly way. This process includes not only the actual writing of code but also the preparation of requirements and objectives, the design of what is to be coded and confirmation that what is developed has met objectives.

Software development Lifecycle (SDLC)

This is a framework for defining tasks performend at each step in the Software Development process.

The SDLC lifecycle goes in this manner:
  • Planning
  • Analyis ans Design
  • Implemetation
  • Integration and Testing
  • Maintainance
    •
Planning Analyis and Design Implementation Integration and Testing Maintainance
Planning entails understanding and finding out what to do, Understanding the scope of the project. Analysis and Design entails How to do what has been documented in the project scope. Here you get the minimum viable product and identify dependences, Use sketches, wireframes to get a view of what needs to be done Implementation entails DOING the project here CODING takes place. Integration and Testing entails checking the task status to see what kind of work was produced. it is important to implement wit the user in mind Mantainance entails overseeing the process flow, adding more functions where necessary
